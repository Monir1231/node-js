# Node.js Learning Collection

This repository contains a complete set of Node.js examples and exercises covering core topics from basic scripting to server-side programming.

## Introduction

You have completed a wide range of Node.js topics, and this repository is organized to demonstrate that progress. It includes examples for modules, file system operations, HTTP servers, streams, events, routing, and package management.

## What’s included

- **Core Node.js fundamentals**: basic modules, `require`, `exports`, and file handling.
- **File System (`fs`)**: reading, writing, and working with files.
- **HTTP and servers**: creating servers, handling requests and responses.
- **Event-driven programming**: using the `events` module and custom event emitters.
- **Streams**: readable, writable, duplex, and transform stream examples.
- **Routing**: basic routing implementations using Node.js.
- **NPM and packages**: package setup, installation, and module usage.
- **Utility and CLI tools**: command-line helpers, encoding/decoding, and small utilities.

## How to use this repository

1. Open a folder from the list below in VS Code.
2. Run `npm install` if a project has a `package.json` file.
3. Start the example with `node index.js` or the relevant entry file.

## Example folders

- `Buffer/` - Buffer handling and binary data examples.
- `CLI Tools/` - Command-line tools and scripts.
- `Encoding and Decoding/` - Text encoding and decoding examples.
- `Event Emitter/` - Event emitter patterns.
- `Implement Basic Routing/` - Simple routing server logic.
- `module/` - Custom module creation and importing.
- `NPM/` - NPM package usage and examples.
- `Server/` - Basic HTTP server setup.
- `simple blog router server/` - A simple blog-style routing server.
- `Steams/` - Stream examples including duplex and transform streams.
- `Update Version/` - Version and package update patterns.

## Running examples

Open any folder and run a Node example file, for example:

```bash
cd "d:\Node js\Server"
node index.js
```

If a folder uses `package.json`, install dependencies first:

```bash
npm install
```
